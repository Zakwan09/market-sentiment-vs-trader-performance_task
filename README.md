# market-sentiment-vs-trader-performance_task
## Objective
Analyze how market sentiment (Fear/Greed) impacts trader behavior and performance.

## Dataset
- Bitcoin Sentiment Data
- Hyperliquid Trader Data

## Steps to Run

1. Install dependencies:
    pip install pandas seaborn matplotlib

2. Place files:
- historical_data.csv
- fear_greed_index.csv

3. Run:
   Task_Round_0_analysis.ipynb


## Outputs
- PnL distribution chart
- Win rate comparison
- Trade size behavior

## Key Findings
- Greed → higher profit but more volatility  
- Fear → stable but lower returns  
- Traders change behavior based on sentiment
